# easter egg clips

Two raw clips for the quest (the same ones `secret()` prints in the run.html console).
Each comes with its `.srt` subtitles (the lyrics), already timed to the clip.

| clip | run it through |
|------|----------------|
| `vocode_clip.mp4` | **Karaoke (local)** |
| `cyberstress.mp4` | **PixelCrash glitch** |

`vocode_clip.mp4` is a 44s cut; `cyberstress.mp4` is the full video.
